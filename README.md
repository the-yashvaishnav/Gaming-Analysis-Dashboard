# Gaming-Analysis-Dashboard

## 📌 Description
This project presents an interactive dashboard to analyze gaming data, including player performance, game popularity, scores, and engagement trends. It helps in understanding user behavior and identifying key insights to improve gaming strategies and overall experience.

## 📊 Tools Used
- Tableau

## 📈 Key Insights
- Total Games
- Top Developers by Number of Games 
- Genre-wise Sales Analysis 
- Top Selling Games by Global Sales
